# Predict_Modul_Flask
Flask приложение для прогноза модуля упругости при растяжении композиционных материалов

Для запуска приложения запустите программный код "modul_predict1.py";

На сервере http://192.168.0.103:5000 заполните форму с известными физико-механическими характеристиками разрабатываемого композитного материала;

Нажмите "Predict";

Рассичтанное значение модуля упругости при растяжении отобразится под строкой "Result"
